# StudyPDF

A personal study tool for students. Load any PDF from your instructor, 
annotate and highlight directly on it, insert rich-text note pages between 
slides, and ask an AI to explain concepts, summarize chapters, and generate 
practice quizzes — all in one place.

## Features
- PDF viewer with highlight, underline, and annotation tools
- Insert Word-style blank pages between any PDF pages
- Rich text editor with full toolbar (bold, bullets, headings, colors, math, tables)
- AI chat panel powered by Gemini 2.0 Flash (explain, summarize, quiz, study tips)
- Cloud sync via Google Drive and OneDrive
- Clean blue & amber UI designed for long study sessions

## Tech stack
- React + Vite (frontend)
- Node.js + Express (backend)
- PDF.js (PDF rendering)
- TipTap (rich text editor)
- Gemini 2.0 Flash API (AI)
- Microsoft Graph API + Google Drive API (cloud sync)
